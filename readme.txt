== Description ==

Create stunning pricing tables in seconds, with drag and drop interface.

* 571 Icons
* Drag and drop (Special thanks to Syamil MJ and it's Aqua Page Builder Plugin)
* Select how many pricing tables you want on a row (1,2,3,4 and 6)
* Customization options for many elements
* Unlimited colors
* Bootstrap 3.0 compatible

Check the [live demo of this plugin](https://herowp.com/hero-pricing-tables-wordpress-plugin-demo/ "HeroWP Pricing Tables Plugin Demo").

== Installation ==

Pretty simple. Just follow this steps

1. Upload folder `herowp-pricing-tables` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. On the left menu in Wordrpress, you will see a menu called HeroWP Pricing Tables.
4. You will find help on how to use it after accessing HeroWP Pricing Tables.

== Frequently Asked Questions ==

= How tu use it? =

Go to HeroWP Pricing Tables, and drag the block "Pricing Tables" on the right area. Open the block configuration, and customize your own details. When finish editing, copy the shortcode from the right (i.e [pricing_layout id="your_id"]), into any page or post .
